## Project Overview

DevRoots Studio v2 is a sophisticated, full-stack ecosystem designed to power a modern digital agency. Beyond a simple public-facing site, it serves as a robust client portal and an innovative "Discord Bot Builder" store. The platform allows users to configure, purchase, and automatically compile custom Discord bots, which are then delivered as downloadable artifacts.

## Technical Excellence

The architecture follows a modular monolithic approach using **NestJS** for a strongly-typed backend and **React** for a high-performance frontend. 

### Key Innovations:
- **Bot Compilation Engine**: A standalone package that resolves feature dependencies and dynamically merges templates to generate production-ready Discord bots.
- **Client-Server State Split**: Utilizing **TanStack Query** for resilient server state and **Jotai** for lightweight, atomic client state.
- **Security First**: Implementation of AES-256-GCM encryption for sensitive credentials and a multi-layered authentication flow via Discord OAuth2.

## The Developer Experience

Built with scalability in mind, the project leverages **Prisma ORM** for type-safe database interactions and **Docker** for consistent deployment environments. The UI is crafted with **Tailwind CSS** and **shadcn/ui**, ensuring a premium, modern aesthetic with smooth interactions powered by **Framer Motion** and **Lenis**.
